# 2b2t Auto Queue Mod

Automatically logs you into whatever server of your choice.

By: aytimothy#0919

Shoutout to the guys at [The Emperium](http://discord.gg/emperium)!

## Building

1. Clone
2. Follow the instructions in [README.txt](README.txt)
3. Build (`gradlew build`)
